Used in Production At Google & Everywhere

Google builds many critical web apps in Dart. The next generation of Google AdWords is built in it. Google Fiber's web app is built in it. Google's internal CRM is built in it.

Outside Google, amazing companies like Blossom, Workiva, and DGLogik have been building their products in Dart.

Easy to learn

{% prettify dart %} var headline = querySelector('#headline'); headline.scrollIntoView(ScrollAlignment.CENTER); headline.classes.add("highlighted"); {% endprettify %}

With a consistent language, well-crafted standard libraries, and cleaner DOM, Dart is a good choice even for programmers with limited or no JavaScript experience. Things work the way C, C#, ActionScript, and Java developers have come to expect.

{% prettify dart %} for (int i = 100; i >= 0; i--) { countdown.text = "Time: $i"; await window.animationFrame; } {% endprettify %}

Dart has one way to manage dependencies, and one way to import them. Its standard libraries have many classes that would otherwise had to be provided by micro-libraries or built from scratch. This makes codebases consistent, with one preferred way to do most things.

{% prettify dart %} countdown.onDoubleClick.listen((ev) { ev.stopPropagation(); restart(countdown); }); {% endprettify %}

Optimizing compilation, tree shaking, minification, and asset transformation are all baked into one build step, configured by one file.

Stable

Google and other companies have been using Dart for years now. Despite hundreds of commits to the SDK each month, the API stays stable.

And because there is a sane wrapper around the DOM and window APIs, you don't need to worry about browser support. Most of that volatility is taken care of for you.

{% img 'screenshot02.png' %}

Made for application programming

Client-side development has challenges that are different from, say, backend development. Dart recognizes that. Client apps have a lot of asynchronicity, so Dart has `async/await`. Clients have many streams (of events), so Dart has a well crafted `Stream` class and generators. OOP is a great fit for UI development, and Dart is purely object-oriented.

Sporting a solid web framework

You can do Dart web development with whatever web framework (or lack of one) you choose. We recommend Angular 2 Dart, which Google uses in production.
